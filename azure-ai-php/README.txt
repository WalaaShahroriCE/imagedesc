README - نشر المشروع على Hostinger (PHP version)
===============================================

ملف المشروع هذا مصمم ليعمل على استضافة Hostinger (Shared hosting) التي تدعم PHP.

الملفات المضمنة:
- index.html         -> الواجهة الأمامية (ارفعها في public_html)
- backend.php        -> سكربت PHP الذي يتصل بـ Azure OpenAI
- .env               -> ملف القيم (ضع بياناتك هنا)
- README.txt         -> هذا الملف

خطوات النشر:
1) افتحي لوحة Hostinger > File Manager.
2) اذهبي إلى مجلد public_html أو المجلد الذي تريدين نشر الموقع فيه.
3) ارفعي الملفات وفكّي الضغط إن لزم.
4) عدّلي ملف .env وضعي فيه قيمك:
   AZURE_ENDPOINT=https://imgdesc-resource.cognitiveservices.azure.com
   AZURE_DEPLOYMENT_NAME=gpt-4o-mini
   AZURE_API_VERSION=2025-01-01-preview
   AZURE_API_KEY=<<انسخي مفتاح Azure هنا>>

ملاحظات مهمة:
- تأكدي أن ملف .env غير قابل للوصول العام عبر المتصفح.
  في Hostinger عادة يتم وضع ملفات التهيئة داخل نفس المجلد ويكون الوصول محمي، لكن يمكنك أيضًا تخزينه خارج public_html إذا رغبت.
- تأكدي من أن cURL مفعّل في استضافة Hostinger (معظم الخطط تكون مفعّلة).
- إذا واجهتِ أي خطأ، فعّلي وضع عرض الأخطاء مؤقتًا أو راجعي سجلات الخادم (logs).

طريقة العمل:
- الواجهة (index.html) تقرأ الصورة كـ Data URL (base64) ثم ترسلها عبر POST إلى backend.php.
- backend.php يرسل صورتك إلى Azure OpenAI ويسترجع وصف قصير ثم أوصاف تسويقية.
- النتيجة تُعاد كـ JSON وتُعرض في الواجهة.

إذا رغبتِ، بقدر أقدملك مساعدة بعد الرفع:
- أتحقق أن .env مضبوط بشكل صحيح.
- أساعدك بتعديل أي خطأ يظهر في السجلات.
- أضبط الحماية بحيث لا يمكن الوصول إلى ملف .env من المتصفح.
